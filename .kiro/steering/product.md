# Product Overview

## 製品名
S3 Tables CLI (s3t)

## 目的
Amazon S3 Tables リソースを簡単に作成・管理するためのコマンドラインツール。

## 対象ユーザー
- AWS S3 Tables を利用するデータエンジニア
- インフラ自動化を行う DevOps エンジニア
- S3 Tables ベースのデータレイクを構築する開発者

## 主要機能
- Table Bucket、Namespace、Table の階層的な作成
- 既存リソースの自動検出とスキップ
- ユーザーフレンドリーなエラーメッセージと解決策の提示
- AWS デフォルト認証チェーンのサポート
- `--profile` オプションによる複数 AWS アカウントの切り替え
- `--region` オプションによるリージョン指定

## ビジネス目標
- S3 Tables リソース作成の手順を簡素化
- 繰り返し作業の自動化による生産性向上
- 一貫したリソース命名とバリデーションの強制

## 設計原則
- シンプルで直感的な CLI インターフェース
- 冪等性のある操作（既存リソースは再作成しない）
- 明確なエラーメッセージと復旧手順の提供
